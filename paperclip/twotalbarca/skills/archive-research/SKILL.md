# Archive Research Workflow

Use this when a story may benefit from historical context.

Steps:
1. Define the present editorial question.
2. Search for directly relevant prior matches, players, seasons, or tactical precedents.
3. Pull only details that can be tied back to the current question.
4. Write one sentence explaining why the archive link matters now.
5. If the connection is weak, recommend no archive insertion.

Standards:
- avoid nostalgia for its own sake
- distinguish documented history from fan memory
- note when external verification is still needed
