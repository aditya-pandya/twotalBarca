# totalBarca Editorial Positioning Decisions

Last updated: 2026-05-02
Status: locked unless Aditya changes it explicitly.

## 1. Locked brand and public-copy rules

Public-facing brand:
- totalBarca

Internal-only codename:
- twotalBarça / twotalbarca

Hard public-copy rules:
- never mention AI on the public-facing site
- never imply the publication is AI-generated or process-driven
- never use the term `blog`
- do not use fake newsroom/status language in public positioning
- do not promise surfaces that are currently dormant

## 2. Locked tagline system

Primary tagline:
- Less noise. More Barça.

Supporting line:
- Match context, club memory, cleaner judgment.

These remain locked.

## 3. Scope pivot note

The 2026-05-02 product change did not change the tagline system.
It did change what the tagline points to.

Active public product now:
- one weekly dispatch with exactly five topics
- minimal last-match and next-match context

It no longer points to:
- a standalone Brief
- article-first publishing
- a broad archive-first publication surface

## 4. Homepage hero deployment

Live wording target:
- Eyebrow: `Less noise. More Barça.`
- Title: `totalBarca`
- Deck: `Match context, club memory, cleaner judgment.`
- Supporting copy: `For supporters who do not need to be shouted at. Start with this week's five-topic dispatch, then use the match capsules to stay oriented around the last match and the next one.`

## 5. Product positioning in plain English

Best framing now:
- a weekly Barça dispatch for supporters tired of daily churn
- emotionally inside the club, intellectually above the noise
- opinionated without becoming loud

## 6. House voice

The house voice should be:
- sharp
- controlled
- football-native
- opinionated, but not shouty
- elegant when earned
- plain when needed

Best one-line rule:
- Write like an editor who watched the week carefully, slept on it, and cut every sentence that was performing certainty instead of earning it.

## 7. Language to lean into

Prefer:
- calm
- clarity
- judgment
- rhythm
- control
- structure
- supporters
- match context
- line of the week

Avoid:
- blog
- content
- viral
- discourse
- newsroom status
- publishing workflow jargon
- daily churn language

## 8. Scope-sensitive copy rule

If public copy mentions what readers do here, it should point to:
- the weekly dispatch
- the last match
- the next match

It should not point to:
- the Brief
- the article stack
- archive shelves
- broad section browsing
