# totalBarca Recovered MVP Roadmap

Last updated: 2026-05-02
Status: active scope summary after the pivot.

## Current public scope

- Home / latest issue
- Dispatch issue page
- Small dispatch archive if needed
- About
- Match context only as a module/capsule within the above

## Guardrails that stay locked

- No public AI/process language
- Never call the product a blog
- Public brand stays `totalBarca`
- Public tagline stays `Less noise. More Barça.`
- Supporting line stays `Match context, club memory, cleaner judgment.`
- No fake newsroom/process/status language in public positioning

## Phase 1 — sharpen the active surface

- keep homepage tight around the weekly dispatch
- enforce the exactly-five-topics contract
- keep last-match and next-match capsules minimal
- keep archive links secondary

## Phase 2 — simplify legacy runtime behavior

- remove stale public references to the Brief
- demote article/archive/analysis/culture surfaces from active IA
- keep dormant routes from reading as active scope

## Future only if explicitly reintroduced

- separate article publishing program
- separate Brief surface
- match notes landing surface
- archive/vault expansion
- topic/person/season browse layers
- reactions/community features
